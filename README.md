# Sales Report Analysis Project
## Project Overview:

This project is an Excel-based sales report used to analyze sales performance over multiple years. It helps understand customer-level sales trends and compare performance across regions, markets, and divisions.
The report is designed in a business reporting format commonly used in real-world sales and finance teams.

## Dataset Structure
Filters
The report includes filter options at the top:
Region
Market
Division
These filters allow users to focus on specific business segments.

## Customer-wise Sales Table
The main table contains:
Customer names,
Net sales values for the years:
2019,
2020,
2021,
Year-over-year comparison between 2021 and 2020.
## All sales values are shown in USD.
Key Insights:
This report helps identify
Customers with increasing or decreasing sales
Year-over-year sales growth or decline
High-performing and low-performing customers
Sales contribution by region, market, and division
Overall business performance trends
